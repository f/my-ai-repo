## Installation

```bash
npm install padding-npm-package
```

## Usage

```javascript
const padding = require('padding-npm-package');

// Add padding to a string
console.log(padding.left('Hello', 10)); // '     Hello'

// Add padding to a number
console.log(padding.right(123, 6, '0')); // '123000'
```
